# Firebat Pro

PowerShell scripts to customize Firefox shortcuts with a bat icon and multi-profile support.

## Files

- `install.ps1` - Finds and modifies Firefox shortcuts (icon + profile args)
- `install-watcher.ps1` - Sets up watcher to run at login (startup shortcut + VBS wrapper)
- `watcher.ps1` - FileSystemWatcher that monitors shortcuts and re-applies settings on change
- `uninstall.ps1` - Restores Firefox shortcuts to defaults
- `firebatpro.ico` - Multi-size ICO (256, 128, 64, 48, 32, 16px)
- `firebatpro.png` - Source PNG with transparency

## Technical Notes

- Scripts use WScript.Shell COM object to modify .lnk files
- Watcher uses System.IO.FileSystemWatcher for change detection
- VBS wrapper runs PowerShell hidden (no console window)
- Icon was generated by Gemini, background removed with Python/Pillow
- `--no-remote` flag enables multiple simultaneous Firefox profiles
- Shortcut locations checked: Desktop, Public Desktop, Start Menu (user + system)

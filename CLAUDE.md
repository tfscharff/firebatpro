# Firebat Pro

Firefox Home profile shortcut with custom bat-themed icon.

## Files

- `Firebat Pro.lnk` - Windows shortcut launching Firefox with the "default-release" profile
- `firebat.ico` - Custom icon (multi-size ICO for Windows)
- `firebat_transparent.png` - Source image with transparency

## Icon Notes

- Icon was generated by Gemini, background removed and resized with Python/Pillow
- ICO contains sizes: 256, 128, 64, 48, 32, 16px
- Content fills ~94% of icon space for consistent sizing with other taskbar icons

## Shortcut Target

```
"C:\Program Files\Mozilla Firefox\firefox.exe" -P "default-release" --no-remote
```
